Quick Start

Up and running in 30 seconds

terminal
$ git clone https://github.com/yourusername/liveindex.git
$ cd liveindex && docker-compose up -d
$ open http://localhost:3000
# That's it. Drop documents in /documents and start querying.

Built for Production

Everything you need to keep AI grounded in reality

Real-time Sync

Edit a document, see it reflected in AI responses within seconds. No batch jobs. No stale answers. File watcher detects changes and re-indexes instantly.

Permission-Aware

Role-based access control built in. Employees see public docs, managers see team docs, admins see everything. Your AI respects your org structure.

Deploy Anywhere

Docker Compose for local dev. Kubernetes for production. Air-gapped environments supported. Your data never leaves your infrastructure.

Open Source

MIT licensed. Fork it, extend it, contribute back. Connector plugins for Google Drive, S3, Notion, Slack, Confluence. Add your own in hours.
